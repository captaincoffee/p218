## Stack

### Back end
 - **ruby** managed with [rbenv](https://github.com/rbenv/rbenv)
   see [netlify current supported versions](https://www.netlify.com/docs/#ruby)
 - **bundler**
 - **jekyll: 3.8.4**

### Front end
 - **bootstrap** + **popper.js**
 - **jquery** + **jquery.cookie**
 - **instantsearch**

See package.json and yarn.lock for versions.
